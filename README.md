# 企业年会大屏抽奖系统

一款专业的企业年会/活动抽奖系统，支持大屏展示、多奖项管理、部门配额分配等功能，完美适配各类企业活动场景。

## 核心特性

### 🎯 基础功能
- **单文件部署**：构建后生成单个 HTML 文件，可直接用浏览器打开，无需服务器
- **大屏展示**：精美的欢迎页、奖项页、抽奖页，支持全屏展示
- **滚动动画**：抽奖时姓名滚动带有加速/减速效果，停止后有撒花庆祝特效
- **多奖项管理**：支持自定义奖项名称、数量、奖品描述
- **数据持久化**：所有数据保存在浏览器本地，刷新不丢失

### 🎨 视觉效果 (2027 级全局奢华美学)
- 呼吸式暗金极光背景动画
- 漫天飞舞的金色星光粒子特效
- 白金金属渐变欢迎大字
- 暗光琉璃 3D 卡片及 Shimmer 金砂扫光流光特效
- 第一眼震撼全场的视觉体验

### 📄 大屏展示模式
- **常规平铺布局**：适合少量中奖者展示（溢出时支持鼠标滚动）
- **谢幕式自动滚动**：平缓匀速自动循环，适合大量中奖名单展示
- **演示翻页模式 (🆕)**：
  - 完美适配物理 PPT 翻页笔与键盘方向键 (`ArrowRight`/`Space`/`PageDown`)
  - 大屏两侧悬浮金边手/键控切换按钮
  - 弹性液态页码指示舱
  - 适合多人中奖舞台演示场景

### 👥 人员管理
- **批量 CSV 导入**：支持快速导入大批量参与人员
- **手动添加/编辑**：支持单独添加或修改人员信息
- **字段支持**：姓名、部门、手机号
- **实时搜索**：按姓名、部门或手机号快速筛选
- **数据统计**：实时显示总人数、已中奖人数、剩余人数

### 🏆 奖项管理
- **自定义奖项**：自由设置奖项名称、中奖人数、奖品描述
- **部门配额分配 (🆕)**：
  - 为每个奖项设置各部门的中奖名额配额
  - 自动校验部门候选人数是否充足
  - 配额不足时智能提醒与自动调整
  - 支持剩余名额在全体参与者中随机抽取
- **快捷操作**：点击选择奖项，双击快速编辑

### 🔐 高级受控模式 (需通过 URL 参数解锁)

通过 URL 参数 `?controlled=1` 解锁高级功能（注意参数要放在 `#` 之前）：

```
index.html?controlled=1#/admin
```

解锁后可使用：
- **必中奖项**：指定某人必中特定奖项
- **分奖项精准黑名单 (🆕)**：单独为每个人配置禁止抽取哪些特定奖项（如禁中一等奖但允许三等奖），提供完全无痕的隐蔽后台控制
- **权重设置**：调整个人中奖概率（1-10）
- **CSV 批量导入高级字段**

### ⚙️ 系统设置
- **自定义标题**：抽奖页主标题、欢迎页主副标题、奖项页标题
- **Logo 上传**：上传公司 Logo 在欢迎页展示（建议透明背景 PNG，不超过 500KB）
- **显示部门名称**：可选择在抽奖滚动和中奖展示时显示所属部门
- **主题切换**：支持明暗主题切换
- **密码保护**：自定义后台管理密码
- **每页显示人数**：翻页模式下可选 12/24/36/48 人每页

### 📊 中奖记录
- **完整记录**：记录每位中奖者的姓名、部门、手机号、奖项、时间
- **图片导出**：一键导出精美中奖名单海报图片
- **记录重置**：支持清空历史中奖记录重新开始

## 快速开始

### 方式一：直接使用（推荐）

1. 下载 `dist/index.html` 文件
2. 用浏览器打开即可使用
3. 访问 `index.html#/admin` 进入后台管理

### 方式二：本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

### 页面结构

| 路径 | 说明 |
|------|------|
| `#/` | 大屏展示页（用于投影/大屏幕） |
| `#/admin` | 后台管理页 |

### 后台管理

默认密码：`appinn`（可在系统设置中修改）

**功能模块**：

| 模块 | 功能说明 |
|------|----------|
| 现场控制 | 切换大屏场景（欢迎页/奖项页/抽奖页）、选择奖项、开始/停止抽奖 |
| 人员名单管理 | 查看/搜索/编辑/删除参与人员 |
| 批量导入 | 通过 CSV 批量导入人员数据、下载导入模板、数据统计、清空数据库 |
| 中奖记录 | 查看历史中奖名单、导出图片、重置记录 |
| 系统设置 | 自定义标题、Logo、展示模式、密码等 |

### CSV 导入格式

**基础格式**（两列）：

```csv
姓名,部门
张三,技术部
李四,市场部
王五,人事部
```

**完整格式**（含手机号）：

```csv
姓名,部门,手机号
张三,技术部,13800138000
李四,市场部,13800138001
```

**高级格式**（需解锁受控模式）：

```csv
姓名,部门,必中奖项(奖项名称),禁止中奖(是/否),权重(1-10)
张三,技术部,一等奖,,
李四,市场部,,是,
王五,人事区,,,5
```

### 大屏操作

支持键盘快捷键：
- `空格键`：开始/停止抽奖
- `方向右键 / 空格 / PageDown`：翻页模式下切换下一页

## 技术栈

- React 19 + TypeScript
- Tailwind CSS v4
- Zustand（状态管理）
- Framer Motion（动画）
- Wouter（路由，支持 hash 模式）
- Vite + vite-plugin-singlefile（单文件构建）
- Radix UI + shadcn/ui（UI 组件库）

## 数据存储

所有数据存储在浏览器本地：

| 存储类型 | 内容 | 说明 |
|---------|------|------|
| localStorage | 参与者、奖项、中奖记录、设置 | 持久保存 |
| sessionStorage | 登录状态 | 关闭浏览器后失效 |

**注意事项**：
- 后台和大屏需在**同一浏览器**中打开才能同步
- 清除浏览器数据会丢失所有记录
- 密码为明文存储，仅作为简单防误触保护
